**Makefile**
-   Για να μεταγλωττισετε όλα τα αρχεία: make
-   Για να τρέξετε κάθε εκτελέσιμο: make runΧ, όπου Χ = 1, 2, 3, 4, 5
    για την άσκηση 1, άσκηση 2 κοκ αντιστοιχα
-   Με make clean διαγράφονται όλα τα .ο και τα εκτελέσιμα στο directory build

**Scripts**
-   Τα αποτελέσματα κάθε script εμφανίζονται στο αρχείο 1_5_benchmark_results.txt
    για την άσκηση 5 κοκ.

**1.1**
-   Για να τρέξετε το πρόγραμμα: *make run1*:
    Εκτελείται για 4 threads και 10^6 βαθμό πολυωνύμου.

-   για να τρέξετε το script: *./benchmark1_1.sh [degree] [threads] [runs]*
    Ο επιθυμητός αριθμός εκτελέσων δεν είναι απαραίτητο να δοθεί, καθώς
    ορίζεται ως default 5.

**1.2**
-   για να τρέξετε το πρόγραμμα: *make run2* 
    αυτό έχει για default τιμές 4 threads και 10^6 επαναλήψεις και τρέχει μία φορά για κάθε αλγόριθμο
    όπου 1 είναι η υλοποίηση με mutex, 2 η υλοποίηση με read/write lock και 3 με atomic operation

-   για να τρέξετε το script: *./benchmark1_2.sh [iterations] [runs]*
    Μπορείτε να περάσετε τα επιθυμητά arguments, ωστόσο το script ορίζει ως default
    10^6 επαναλήψεις και 5 εκτελέσεις για κάθε αλγόριθμο. Ο κάθε αλγόριθμος τρέχει
    από πέντε φορές για 1, 2, 4 και 8 threads.

**1.3**
-   για να τρέξετε το πρόγραμμα: *make run3* 
    αυτό έχει για default μέγεθος πίνακα 10^7

-   για να τρέξετε το script: *./benchmark1_3.sh [size of array] [runs]*
    Μπορείτε να περάσετε τα επιθυμητά arguments, ωστόσο το script ορίζει ως default
    μέγεθος πίνακα 10^7 και 5 εκτελέσεις. 

**1.4**
-   για να τρέξετε το script: *./benchmark1_4.sh <number_of_accounts> <transactions_per_thread>*
    *<percentage_of_read_transactions> <number_of_threads> [iterations]*
    Το script ορίζει ως default 5 εκτελέσεις, οπότε είναι προαιρετικό να περαστούν
    ως όρισμα.

**1.5**
-   για να τρέξετε το πρόγραμμα: *make run5* 
    αυτό έχει για default αριθμό threads 4 και 10^6 επαναλήψεις και τρέχει 
    και τα τρία προγράμματα με τα ορίσματα αυτά.

-   για να τρέξετε το script: *./benchmark1_5.sh [iterations] [runs]*
    με default επαναλήψεις 10^6 και 5 εκτελέσεις για κάθε πρόγραμμα.
    Εκτελείται για 1, 2, 4 και 8 νήματα αντίστοιχα.
