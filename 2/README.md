Με make γίνονται compile όλα τα προγράμματα και δημιουργούνται τα εκτελέσιμα στο 
build directory

**2_1**
- Για να τρέξετε το πρόγραμμα: ./build/2_1 <Degree of polynomial> <Number of threads>
- Για να τρέξετε το script: ./benchmark2_1.sh [degree of polynomial] [threads] [number of executions]
    Το script, έχει προκαθορισμένο αριθμο επαναλήψεων 5 για τον υπολογισμό του μέσου όρου, οπότε δεν είναι απαραίτητο να δωθει ως ορισμα.
    Τα αποτελέσματα αποθηκεύονται στο αρχείο 2_1_benchmark_results2.txt 

**2_2**
 - Για να τρέξετε το πρόγραμμα: ./build/2_2 <number of rows> <percentage of 0 elements> <iterations> <threads> <Y for non-uniform distribution>
   η τελευταία παράμετρος είναι προαιρετική και αν δεν δοθεί τότε θεωρείται ότι η κατανομή των μη μηδενικών στοιχείων είναι ομοιόμορφη.
 - Το Script είναι ένα Python προγραμμα που δημιουργεί αρχεία και csv files με τα αποτελέσματα των εκτελέσεων στον φακελο bench_results.
   Δεν χρειάζεται να δωθούν παράμετροι στο script. Δημιουργεί διαγράμματα για σταθερες παραμέτρους ενώ αλλάζουν όσες μας ενδιαφέρουν.
   Επίσης επειδή το scheduling έχει οριστεί ως runtime αν δεν τρέξετε το script που φροντίζει για την κατάλληλη ενημέρωση τημεταβλητής περιβάλλοντος OMP_SCHEDULE
   πρέπει να το ορίσετε εσείς πριν την εκτέλεση του προγράμματος. Τα πειράματα γίνονται με Static (και guided στην περίπτωση μη ομοιόμορφης κατανομής).


**2_3**
- Για να τρέξετε το πρόγραμμα: ./build/2_3 <size of array> <serial or parallel> <number of threads>
- Για να τρέξετε το script: ./benchmark2_3.sh [size of array] [number of executions]
    Το script, έχει προκαθορισμένο array size 10^7 και 5 εκτελέσεις για τον υπολογισμό
    του μέσου όρου, οπότε δεν είναι απαραίτητο να δοθούν τα ορίσματα αυτά. Επιπλέον,
    τρέχει το πρόγραμμα με σειριακό τρόπο αλλά και παράλληλο για αριθμό threads 1, 2, 4 και 8
    αντίστοιχα.
